Chiru WebClient

Chiru web client implementation for realXtend Tundra

View project onGitHub

This JavaScript library implements a light-weight web client currently for realXtend Tundra server and it needs a WebSocketModule add-on installed into the Tundra server so a connection through a WebSocket can be established. This project is still work in progress, but the goal is to provide a simple and light-weight library and API for creating a customized web client for Tundra and also for other similar virtual world servers.
